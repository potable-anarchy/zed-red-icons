# Zed Red Icons

An ominous red icon theme for Zed editor that complements the Zed Red color theme. Features crimson-tinted icons with dark backgrounds for a cohesive bloody aesthetic.

## Installation

1. Copy the `icon_themes/` and `icons/` directories to your Zed extensions folder
2. Open Zed settings and select "Zed Red Icons" as your icon theme
3. Or manually add to your `settings.json`:
   ```json
   {
     "icon_theme": "Zed Red Icons"
   }
   ```

## Features

- Comprehensive icon coverage for popular file types
- Crimson red color palette (#dc143c) matching the Zed Red theme
- Dark folder and chevron icons
- Support for programming languages, config files, media files, and more
